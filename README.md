# India Consumer Price Index (CPI) Dashboard

## Overview
An interactive Excel dashboard analyzing India's Consumer Price Index (CPI) and inflation trends from 2019–2025.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Workbook Structure
- **Raw_Data** – Original CPI dataset.
- **Clean_Data** – Cleaned and transformed data with additional calculated columns.
- **Pivot_Table** – Pivot Tables used for KPIs, charts, and slicers.
- **Dashboard** – Interactive dashboard with KPI cards, charts, and slicers.
- **Business_Insights** – Executive summary, key insights, recommendations, and conclusion.

## Dashboard Features
- KPI Cards
- Monthly Inflation Trend
- Category-wise Average CPI
- Area-wise Average CPI
- Year-wise Average CPI
- Interactive Slicers
